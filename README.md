You need python 3.12 or 3.13 installed on your computer for this lab

Install the pre-requisites
1. pip install ydata-sdk
2. pip install streamlit
3. pip install "streamlit[pdf]"

Now you can run

streamlit run app.py

Ignore the warnings in the console

Follow the instructions in UI that opens in browser.

Use Control-C to stop the streamlit app in the console.

A sample data file named distract.csv is included in the repo.

Optionally you can setup a virtual environment for ydata 

python3.12 -v venv ydata

source ydata/bin/activate

You should see (ydata) in the prompt, now follow the above instructions to install dependencies within this virtual environment.
